# openclawbot

Openclaw's workspace and configuration files. This repo is read/written by the openclaw bot via GitHub App authentication.

## Structure

- `SOUL.md` - Core personality and values
- `IDENTITY.md` - Name, creature type, vibe
- `AGENTS.md` - Workspace rules and conventions
- `BOOTSTRAP.md` - First-run setup guide
- `HEARTBEAT.md` - Periodic task checklist
- `TOOLS.md` - Local environment notes
- `USER.md` - Info about the human
- `canvas/` - Interactive canvas UI
